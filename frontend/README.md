# Aorta 3D Viewer

Visualizador interactivo 3D para modelos médicos de aorta abdominal en formato STL y VTP.

## Características

- 📁 Carga de archivos STL y VTP
- 🎨 Visualización 3D interactiva con Three.js
- 🖱️ Controles de rotación y zoom con ratón
- 💨 Interfaz moderna con Tailwind CSS
- 📱 Responsive design
- ⚡ Construido con Vite y React

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

El servidor de desarrollo se abrirá en `http://localhost:5173`

## Build

```bash
npm run build
```

## Uso

1. Haz clic o arrastra un archivo STL/VTP al área designada
2. El archivo se visualizará en 3D en el visor
3. Usa el ratón para rotar el modelo
4. Usa la rueda del ratón para hacer zoom

## Tecnologías

- **React** - UI Framework
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Three.js** - 3D graphics
